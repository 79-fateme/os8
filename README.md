Chmode options:

f: silent , quiet 

بیشتر پیام های خطا را سرکوب می کند

v: verbose

  برای هر فایلی که پردازش می شود یک عیب یابی صادر می کند

R: recursive 

فایل ها و دایرکتوری ها را به صورت بازگشتی تغییر می دهد
